# Email Automation from Google Sheets

## 📌 Description

This automation sends an email whenever a new row is added to Google Sheets.

## ⚙️ Tools

* Make
* Google Sheets
* Email module

## 🔄 Workflow

1. New row added in Google Sheets
2. Data is extracted (Name, Email, Message)
3. Email is sent automatically

## 💡 Use Case

* Lead follow-up
* Notifications
* Simple CRM automation

## 🚀 Result

Fully automated email sending without manual work
